Batch costing (CSV)

Upload a CSV with a header row of parameter names (any of: name, layerCount, boardW, boardH, pcbType, material, via, trace, finish, quality, region, orderQty, …). Each row is costed with unspecified parameters taken from the current configuration.

Quote calibration teach the model your real prices

Log actual supplier quotes against the model's prediction for the same configuration. The model learns a per-region correction factor (median ratio) and shows its live accuracy. This is how the accuracy claim is earned, not asserted.
