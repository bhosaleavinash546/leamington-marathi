CostVision — Help Centre

Everything you need to master AI-powered should-costing

1
Select a Commodity

Choose the manufacturing process from the tabs at the top of the input panel. CostVision supports 20+ commodity types including Machining, Casting, Injection Moulding, Sheet Metal, PCBA, Composites, Wiring Harness, and more.

2
Enter Part Data
  • Fill in the Part Name at the top of the form.
  • Enter material, operations, and process-specific parameters.
  • Set Volume, Overhead %, and Margin % in Universal Costs.
  • Optionally upload a part photo — it appears on the PDF report.
3
Select Region & Currency

Use the Region selector in the top bar to switch manufacturing location. All machine rates, labour rates, and material prices adjust automatically for the selected country. Use the Currency dropdown to display costs in your preferred currency.

4
Click Calculate

Press Calculate to run the 8-bucket cost model: Raw Material → Process → Labour → Tooling → Packaging → Logistics → Overhead → Margin. Results appear in the right panel with interactive charts, sensitivity analysis, and AI-powered insights.

5
Export & Save
  • Export Excel — 6-sheet workbook with full cost breakdown and rate buildup.
  • PDF Report — 7-section professional report with AI insights.
  • Save Scenario — store configurations and compare multiple designs side-by-side.
✦
What is the AI Agent?

The AI Agent tab uses Claude (Anthropic) to understand natural language descriptions of your part and automatically populate the cost model. Simply describe what you need costed — the AI handles the rest.

→
How to Use the AI Agent
  1. Click the AI Agent ✦ tab.
  2. Type a description, e.g.: "Aluminium 6061 bracket, 0.8kg, CNC milled, 3 operations, 50k annual volume, UK manufacture"
  3. Optionally upload a part photo or CAD file.
  4. The AI will auto-fill material, machine selections, cycle times, and volumes.
  5. Review the suggestions, adjust if needed, then click Calculate.
⚠
AI Validation

CostVision validates all AI-returned IDs against the rate library. If the AI suggests a material or machine that doesn't exist in the library, it will be flagged with a warning toast and nulled out — you'll need to select manually.

⚡
Tips for Better Results
  • Mention specific alloys: "Al 6082-T6", "Grade 304 stainless"
  • Include annual volume for accurate tooling amortisation
  • Specify the manufacturing country for regional rate accuracy
  • Add surface finish requirements (anodising, powder coat, painting)
📁
Supported File Types
  • Photos: JPG, PNG, HEIC — uploaded via the Part Photo zone (always visible)
  • CAD for AI analysis: PNG/JPG screenshots of STEP, DXF, or SolidWorks files via the AI CAD ✦ tab

How do I change the manufacturing region?
Use the 🌍 dropdown in the top navigation bar. Selecting a different country automatically adjusts machine rates, labour rates, energy costs, and material transport costs for that region. All rates are from the Rate Library which you can inspect and edit via Edit Rates.
Can I edit the rate library?
Yes. Click Edit Rates in the action bar. You can modify machine hourly rates, labour rates, and material prices. Changes apply immediately to the next Calculate run. Use Reset to Defaults to revert to the original published rates.
What is the 8-bucket cost model?
CostVision computes: Raw Material + Process + Labour + Tooling + Packaging + Logistics + Overhead + Margin. This mirrors the aPriori / Siemens Teamcenter Sourcing should-cost methodology used by Tier 1 automotive and aerospace OEMs.
What is Wright's Law / Learning Curve?
Wright's Law states that for every doubling of cumulative production volume, unit cost decreases by a fixed percentage (typically 10–20% for manufacturing). Enable the Apply learning curve checkbox and set the curve % (default 85% = 15% reduction per doubling). Set your annual volume and reference volume for the calculation.
How do I compare two designs?
Save each design as a scenario using Save Scenario. Then click the Scenarios tab in the results panel, select two scenarios, and click Compare. A delta breakdown table shows cost differences by bucket, with colour-coded savings/increases.
Are my scenarios saved if I close the browser?
Yes — CostVision uses IndexedDB for persistent browser storage. Scenarios survive page reloads and browser restarts. Use Export Scenarios to create a portable JSON backup file. If IndexedDB storage is full or unavailable, a warning toast will appear.
OEE
Overall Equipment Effectiveness — uptime × performance × quality. Typical range 60–95%.
Cycle Time
Time per machine cycle in hours. Includes machining + handling. Parts per cycle divides this.
Manning
Number of operators per machine. Affects labour cost per part.
MOQ
Minimum Order Quantity — minimum batch a supplier will accept per purchase order.
NRE
Non-Recurring Engineering cost — one-time tooling, setup, or design cost.
PPV
Purchase Price Variance — % difference between quoted supplier price and the should-cost model.
Amortisation
Spreading tooling NRE cost over the expected production volume to compute per-part tooling cost.
Casting Yield
Ratio of finished part weight to total pour weight (runner + gate loss). Typically 0.6–0.95.
Labour Efficiency
Effective labour utilisation (0–1). Accounts for non-value-added time. Typical: 0.80–0.95.
DfM Score
Design for Manufacturability score (0–100) from AI CAD analysis. Higher = easier to manufacture.
Tornado Chart
Sensitivity analysis chart showing which input variables have the largest impact on total cost.
Wax Recovery
In investment casting, fraction of wax recovered via autoclave dewaxing. Default 80%.
!
"Scenario could not be saved" warning

Your browser's IndexedDB storage may be full or blocked. Go to browser settings → Site Data → clear storage for this site, then reload. Alternatively, export your scenarios to JSON before clearing.

!
AI Agent returns invalid material/machine IDs

The AI may suggest IDs not in the current rate library. Invalid IDs are automatically nulled and a warning toast is shown. Manually select the correct material or machine from the dropdown after the AI populates other fields.

!
PDF export is missing the Insights section

The §7 AI Insights section only appears if the AI Insights tab has generated recommendations (click Calculate, then view the Insights tab to trigger generation). Run Calculate before exporting the PDF.

!
Rate library shows "stale" warning

The rate library data is over 90 days old. Click Edit Rates to review and update material prices, machine rates, and labour rates to current market values.

!
Results show unexpectedly high material cost

Check the Material Utilisation field — a value less than 1 means significant scrap allowance. For casting, check Casting Yield and Reject Rate. Verify the Consumables cost per part is not accidentally large.
