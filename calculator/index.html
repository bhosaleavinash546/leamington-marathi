How do I change the manufacturing region?
Use the 🌍 dropdown in the top navigation bar. Selecting a different country automatically adjusts machine rates, labour rates, energy costs, and material transport costs for that region. All rates are from the Rate Library which you can inspect and edit via Edit Rates.
Can I edit the rate library?
Yes. Click Edit Rates in the action bar. You can modify machine hourly rates, labour rates, and material prices. Changes apply immediately to the next Calculate run. Use Reset to Defaults to revert to the original published rates.
What is the 8-bucket cost model?
CostVision computes: Raw Material + Process + Labour + Tooling + Packaging + Logistics + Overhead + Margin. This mirrors the aPriori / Siemens Teamcenter Sourcing should-cost methodology used by Tier 1 automotive and aerospace OEMs.
What is Wright's Law / Learning Curve?
Wright's Law states that for every doubling of cumulative production volume, unit cost decreases by a fixed percentage (typically 10–20% for manufacturing). Enable the Apply learning curve checkbox and set the curve % (default 85% = 15% reduction per doubling). Set your annual volume and reference volume for the calculation.
How do I compare two designs?
Save each design as a scenario using Save Scenario. Then click the Scenarios tab in the results panel, select two scenarios, and click Compare. A delta breakdown table shows cost differences by bucket, with colour-coded savings/increases.
Are my scenarios saved if I close the browser?
Yes — CostVision uses IndexedDB for persistent browser storage. Scenarios survive page reloads and browser restarts. Use Export Scenarios to create a portable JSON backup file. If IndexedDB storage is full or unavailable, a warning toast will appear.
OEE
Overall Equipment Effectiveness — uptime × performance × quality. Typical range 60–95%.
Cycle Time
Time per machine cycle in hours. Includes machining + handling. Parts per cycle divides this.
Manning
Number of operators per machine. Affects labour cost per part.
MOQ
Minimum Order Quantity — minimum batch a supplier will accept per purchase order.
NRE
Non-Recurring Engineering cost — one-time tooling, setup, or design cost.
PPV
Purchase Price Variance — % difference between quoted supplier price and the should-cost model.
Amortisation
Spreading tooling NRE cost over the expected production volume to compute per-part tooling cost.
Casting Yield
Ratio of finished part weight to total pour weight (runner + gate loss). Typically 0.6–0.95.
Labour Efficiency
Effective labour utilisation (0–1). Accounts for non-value-added time. Typical: 0.80–0.95.
DfM Score
Design for Manufacturability score (0–100) from AI CAD analysis. Higher = easier to manufacture.
Tornado Chart
Sensitivity analysis chart showing which input variables have the largest impact on total cost.
Wax Recovery
In investment casting, fraction of wax recovered via autoclave dewaxing. Default 80%.

Overview — Injection Moulding

Injection moulding is a high-volume thermoplastic process. A heated screw injects molten polymer into a steel or aluminium tool. Cycle times are typically 15–90 seconds. Cost is dominated by tooling amortisation at low-to-medium volumes and material at high volumes.

Inputs Required
  • Material grade (PP, ABS, PC/ABS, PA66-GF30, etc.) and part weight (kg)
  • Projected area (cm²) and tool type (single/multi-cavity, hot or cold runner)
  • Machine tonnage (determined by projected area × cavity pressure)
  • Cycle time (s), OEE (%), cavities per shot
  • Tooling cost (£) and amortisation volume
  • Surface finish and tolerance class (affects multiplier)
  • Annual volume (pcs) and manufacturing region
Step-by-Step Workflow
  1. Select Injection tab or use ⊕ New Costing → Plastics tile.
  2. Choose the material grade and enter part weight. The tool auto-prices material from the rate library.
  3. Enter projected area and cavity count. Machine tonnage is auto-recommended.
  4. Enter cycle time (shoot time + cooling + eject). A typical 0.3 kg part runs 35–55 s.
  5. Enter OEE (typically 75–90%), Manning (usually 0.25–0.5 per machine), and parts per cycle.
  6. Enter tooling cost and amortisation volume. For 50k ppa, a single-cavity ABS tool is typically £18–45k.
  7. Set annual volume, overhead %, margin %, and region, then click Calculate.
Tips
  • Add a 1.08–1.15× surface finish multiplier for textured or Class A surfaces.
  • Multi-cavity tools reduce per-part cost but increase tooling NRE — break-even is usually above 30k ppa.
  • Hot runners reduce cycle time and scrap but add £8–25k to tooling cost.
Common Mistakes
  • Forgetting runner/gate scrap — add 5–15% material uplift for cold runner tools.
  • Under-estimating cycle time for thick-wall or large parts (rule: 1 s per mm wall thickness minimum).
  • Using a 1-cavity assumption when the supplier will almost certainly use 2-cavity for volumes above 20k.
Overview — Sheet Metal Fabrication

Sheet metal fabrication covers laser cutting, turret punching, press brake bending, and optional welding. It is a low-tooling, high-flexibility route suited to low-to-medium volumes (1–50k). Cost is labour-intensive and driven by machine time, bends, and weld length.

Inputs Required
  • Material grade (DC01, S355, 304 SS, 5052 Al) and sheet thickness (mm)
  • Blank dimensions (mm × mm) and part weight (kg)
  • Blanking method: Laser (Fiber / CO₂), Turret punch, or Shear
  • Number of bends and bend length (mm) for press brake
  • Welding: type (MIG, TIG, spot), length (mm), and number of joints
  • Surface treatment (powder coat, galvanise, primer)
  • Annual volume and manufacturing region
Step-by-Step Workflow
  1. Select SM Fab ✓ tab or use ⊕ New Costing → Sheet Metal tile.
  2. Choose material grade and enter sheet thickness. The process advisor recommends a blanking method.
  3. Enter blank size and part weight. Material cost is auto-calculated including nesting utilisation.
  4. Select laser machine (fiber for stainless / non-ferrous; CO₂ for mild steel below 8 mm).
  5. Enter cut perimeter (mm) — the tool calculates cut time from machine speed.
  6. Add press brake operations: number of bends, bend length, setup time.
  7. Add weld operations if required (MIG seam / spot weld / TIG).
  8. Set volume, overhead, margin, and click Calculate.
Tips
  • Fiber laser is 3× faster than CO₂ for stainless and aluminium — always choose fiber for these materials.
  • Spot welding is 5–10× cheaper than MIG for overlapping flanges — recommend DFM change if possible.
  • Powder coating adds £1.50–4.00/part for small brackets; include it in universal costs.
Common Mistakes
  • Forgetting pierce time for hole features — each pierce adds 1–3 s on a laser.
  • Under-estimating sheet nesting efficiency (typically 65–80%); use 0.72 as default material utilisation.
  • Ignoring setup time for small batches — a press brake setup is 20–45 min regardless of volume.
Overview — Castings (HPDC / Sand)

Casting fills a die or mould with molten metal. HPDC (High Pressure Die Casting) uses permanent steel dies for aluminium/zinc at high volumes. Sand casting uses expendable moulds for iron/steel/aluminium at lower volumes or large parts. Cost is driven by die cost, cycle time, and yield.

Inputs Required
  • Alloy (ADC12, A380, GJL250, EN-GJS-400) and process (HPDC, sand, gravity, investment)
  • Part weight (kg) and casting yield (typically 0.70–0.95 for HPDC)
  • Machine tonnage (HPDC: use projected area × 600 bar / 1000 as starting point)
  • Cycle time (s), number of cavities, OEE (%)
  • Die/pattern cost (£) and amortisation volume
  • Reject rate (%), post-cast operations (shot blast, impregnation, heat treat)
  • Annual volume and region
Step-by-Step Workflow
  1. Select Casting tab or ⊕ New Costing → Castings tile.
  2. Choose alloy and casting process. The model sets appropriate material price and yield defaults.
  3. Enter part weight and casting yield. Poured weight = part weight ÷ yield.
  4. Select machine tonnage. For HPDC: typical 800T die cast produces 2–6 kg parts.
  5. Enter cycle time (HPDC typical: 25–60 s; sand cast: 5–30 min per pour).
  6. Enter die cost. HPDC dies: £80–350k; sand cast patterns: £5–40k.
  7. Enter reject rate (HPDC: 2–5%; sand: 5–12%) and post-cast treatment costs.
  8. Click Calculate to get the full as-cast cost breakdown.
Tips
  • For cast + machined parts, use the Cast+Machine ✓ tab which combines both in one model.
  • HPDC at 2-cavity doubles output without proportionally doubling die cost — check cavity economics above 40k ppa.
  • Investment casting is more expensive per part but achieves ±0.1 mm tolerances without machining.
Common Mistakes
  • Confusing part weight with poured weight — always model poured weight for material cost.
  • Ignoring heat treatment cost for structural aluminium castings (T5/T6 adds £0.40–1.20/kg).
  • Underestimating HPDC die life — budget for 1 die replacement per 100k shots for aluminium.
Overview — CNC Machining

CNC machining removes material from bar, billet, or casting stock using turning centres, VMCs (3/4/5-axis), and grinders. Cost is dominated by machine hourly rate × cycle time. Material waste (buy-to-fly ratio) can be significant for aerospace or complex geometries.

Inputs Required
  • Material (Al6061-T6, 304 SS, Ti-6Al-4V) and billet/bar weight (kg)
  • Finished part weight (kg) — determines buy-to-fly ratio
  • Operations list: machine type, cycle time (h), OEE, Manning
  • Tooling cost per operation (cutting tools, fixtures)
  • Annual volume and region
Step-by-Step Workflow
  1. Select Machining tab or ⊕ New Costing → Machining tile.
  2. Choose material grade. Enter billet weight (raw input) and finished part weight.
  3. Add Operation 1: select machine (VMC 3-axis, Turn/Mill, 5-axis DMG), enter cycle time and OEE.
  4. Add further operations as required (e.g., Op 2: drill and tap on HMC).
  5. Enter per-operation tooling consumable cost (typical: £0.50–3.00 per part for Al, £2–8 for steel).
  6. Set Manning ratio (usually 0.25–1.0 depending on automation level).
  7. Set volume, overhead, margin, region, and click Calculate.
Tips
  • UK 3-axis VMC rates: £55–85/hr; 5-axis: £90–140/hr. Use Edit Rates to customise.
  • For Al parts, OEE of 82–88% is typical. For titanium, expect 65–75% due to tool change frequency.
  • Buy-to-fly above 3:1 signals a DFM opportunity — consider near-net casting or forging.
Common Mistakes
  • Forgetting setup amortisation — a complex 5-axis fixture setup can add £0.50–2.00/part at 10k volume.
  • Using finished weight instead of billet weight for material cost calculation.
  • Assuming 100% OEE — always apply real-world OEE (75–90%) to avoid under-costing.
Overview — Forgings

Forging plastically deforms metal (hot or warm) into a die to produce high-strength near-net-shape parts. Closed-die forging produces tighter tolerances than open-die. Cost is driven by material, press tonnage, die cost, and heat treatment. Forgings have superior grain structure vs. castings.

Inputs Required
  • Material (Al6082, 4340 Steel, Ti-6Al-4V, Inconel 718)
  • Billet weight (kg) and finished forging weight (kg) — flash loss typically 10–25%
  • Process (open-die, closed-die, near-net-shape) and press tonnage
  • Number of blows/strikes, die cost (£), amortisation volume
  • Heat treatment requirement (T4, T6, normalise, quench+temper)
  • Annual volume and region
Step-by-Step Workflow
  1. Select Forging tab or ⊕ New Costing → Forgings tile.
  2. Choose material and enter billet weight. Add flash allowance (15% typical for closed-die Al).
  3. Select press tonnage. Rule of thumb: projected area (cm²) × 3 = tonnes required for Al; ×5 for steel.
  4. Enter number of strikes (typically 3–6 for complex closed-die). Each strike has a machine cycle time.
  5. Enter die cost: Al forgings typically £40–120k; steel £70–180k. Die life: 20–80k shots.
  6. Add heat treatment if required (T6 for aluminium adds £0.60–1.50/kg).
  7. Calculate and review tooling amortisation — major cost driver at low volumes.
Tips
  • Forging vs. casting: forgings are 20–40% stronger but 30–80% more expensive per part — justify with fatigue or safety requirements.
  • Near-net-shape forgings can eliminate machining steps, reducing total cost by 15–25%.
  • China forging: 35–55% lower piece price but factor in 8–12 week lead time and quality risk.
Common Mistakes
  • Forgetting flash trimming cost — typically 10–20% of forging cycle cost for closed-die parts.
  • Under-sizing the press — always add 20% safety margin to minimum required tonnage.
  • Ignoring die maintenance cost at high volumes — budget 15–20% of die cost for lifetime maintenance.
Overview — PCB Fabrication

PCB fabrication converts a bare laminate into a circuit board via lamination, drilling, plating, etching, solder mask, and surface finish. Cost is driven by layer count, board area, HDI complexity, surface finish, and quality grade. Use the 🔬 Image-to-BOM feature to auto-extract specs from a photo.

Inputs Required
  • Board dimensions (mm × mm) and layer count (2, 4, 6, 8+)
  • Technology type (FR4 standard, High-Tg FR4, HDI, Rigid-Flex, RF/Rogers)
  • Surface finish (HASL, HASL-LF, ENIG, OSP, ENEPIG)
  • Via counts: through, blind, buried, micro-vias
  • HDI structure (none, 1+N+1, 2+N+2, any-layer)
  • Quality grade (consumer, industrial, automotive Gr.2, Gr.1, aerospace)
  • Annual volume (panels) and region
Step-by-Step Workflow
  1. Select PCB Fab 🔬 tab or ⊕ New Costing → PCB tile.
  2. Optional: upload a PCB photo via the 🔬 Image-to-BOM zone — the AI auto-fills all fields.
  3. Enter board dimensions and layer count. Each pair of inner layers adds 35–55% to fab cost.
  4. Select technology type. HDI costs 2.5–4× standard FR4 due to laser drilling of microvias.
  5. Select surface finish. ENIG is standard for automotive; HASL-LF for consumer cost reduction.
  6. Enter via counts. Blind/buried vias add 20–60% cost uplift vs. through-vias only.
  7. Select quality grade. Automotive Gr.1/2 adds 25–80% to base fab cost for testing and traceability.
  8. Click Calculate for per-panel and per-board cost with min/mid/max range.
Tips
  • PCB cost scales with area² — halving the board size reduces fab cost by 60–70%.
  • ENIG is worth the premium over HASL for fine-pitch BGAs and SMD pads below 0.5 mm pitch.
  • Panel utilisation (default 75%) — tighter layouts increase this and reduce per-board cost.
Common Mistakes
  • Confusing layer count with copper layer count — a 4-layer board has 2 inner + 2 outer layers.
  • Selecting ENEPIG when ENIG is sufficient — ENEPIG costs 40–60% more and is only needed for wire bonding.
  • Forgetting flying probe or ICT test cost — add £0.15–0.60 per board for automotive quality grades.
Overview — PCBA Assembly

PCBA assembly involves SMT placement (solder paste → pick-and-place → reflow), through-hole soldering (wave or selective), and optional processes (BGA underfill, conformal coat, ICT, X-ray). Cost is driven by component count, placement complexity, BGA presence, and quality grade.

Inputs Required
  • BOM: component type, quantity, unit price, automotive grade flag
  • SMT placements (total qty of all SMT components)
  • Through-hole joints (qty × pin count for THT components)
  • BGA count (drives X-ray requirement)
  • Assembly complexity (low/medium/high/very high)
  • Reflow sides (1 or 2), conformal coat area (cm²)
  • Quality grade and ICT test time (s)
Step-by-Step Workflow
  1. Select PCBA 🔬 tab or ⊕ New Costing → PCBA tile.
  2. Optional: upload a PCB photo to auto-extract BOM via AI Image-to-BOM.
  3. Review or enter the BOM. Mark any automotive-grade components (3–5× price premium).
  4. Enter assembly parameters: SMT placements, TH joints, BGA count.
  5. Select complexity (high = fine-pitch QFN/BGA, double-sided, tight spacing).
  6. Add ICT test time (s) — automotive Gr.2 minimum is 90 s; Gr.1 is 180+ s.
  7. Enter conformal coat area if applicable (automotive standard IPC-CC-830).
  8. Click Calculate for per-board PCBA cost split by BOM, SMT, and test.
Tips
  • CPH (components per hour) for a modern high-speed SMT line: 40,000–80,000 CPH.
  • BGA rework is £8–25 each — minimise BGA count or specify sockets for prototypes.
  • AEC-Q100/Q200 automotive parts cost 3–8× the consumer equivalent — always flag these in the BOM.
Common Mistakes
  • Forgetting NRE (stencil, programming, first article inspection) — typically £350–800 one-time.
  • Using consumer IC pricing for automotive designs — always verify AEC-Q grade pricing.
  • Under-stating placement count — count every pad on every SMT component including decoupling caps.
Overview — Wiring Harness

A wiring harness bundles multiple wire circuits into a routed assembly with connectors, terminals, splices, conduit, and tape. Cost is labour-intensive (60–75% of total cost for simple harnesses). Assembly is largely manual, making labour rate and assembly time the primary cost drivers.

Inputs Required
  • Wire circuits: gauge (mm²), length per circuit (m), insulation type
  • Connector types and pin counts (Delphi MX150, TE AMP, Molex, etc.)
  • Splices, seals, and terminal count
  • Conduit/corrugated tube length (m) and tape area (m²)
  • Assembly time estimate (hours per harness)
  • Test type (continuity, HiPot, functional)
  • Annual volume and manufacturing region
Step-by-Step Workflow
  1. Select Harness ✓ tab or ⊕ New Costing → Wiring Harness tile.
  2. Add wire circuits: for each circuit, enter gauge, length, and insulation type.
  3. Add connectors: housing unit price, terminal count, and seal cost.
  4. Enter conduit length and tape area. Tape is typically 120% of circuit length.
  5. Enter assembly time (h). A 10-circuit door harness typically takes 0.25–0.45 h to assemble.
  6. Select test type. HiPot test adds £0.05–0.15 per harness; functional adds £0.20–0.60.
  7. Set region — harness assembly in Romania/Morocco is 40–60% lower labour cost than UK.
  8. Click Calculate for material, assembly, and test cost breakdown.
Tips
  • For automotive harnesses, use 0.35–0.5 mm² for signal circuits and 1.5–4.0 mm² for power circuits.
  • Splice usage: each splice saves a connector pin but adds crimping time — net saving only above 5k volume.
  • Romania and Morocco rates are £3.50–5.50/hr — benchmark for all volume automotive harness work.
Common Mistakes
  • Forgetting terminal crimp time — each crimp adds 4–8 s assembly time.
  • Under-estimating tape usage — use 1.2× circuit length for full wrap, 0.6× for partial wrap.
  • Ignoring drawing/BOM preparation NRE — a complex harness drawing takes 4–16 engineering hours.
Overview — Cast + Machine

Cast+Machine combines HPDC (or sand/gravity) casting with post-cast CNC machining in a single integrated model. This is the correct approach for any cast part that requires subsequent machining for critical dimensions, threads, or sealing faces. Do not model these separately.

Inputs Required
  • Casting: alloy, process, poured weight, die cost, machine tonnage, cycle time
  • Post-cast: heat treatment (T5, T6, HT, none), shot blast, impregnation
  • Machining operations: machine type, cycle time, OEE, tooling cost per op
  • Annual volume and region
Step-by-Step Workflow
  1. Select Cast+Machine ✓ tab or ⊕ New Costing → Cast + Machine tile.
  2. Complete the Casting section: alloy, process, poured/part weight, die cost, cycle time.
  3. Add post-cast operations: heat treatment (T6 for structural Al: +£0.80–1.50/kg), shot blast (£0.10–0.30/part), impregnation (£0.25–0.60/part).
  4. Add machining operations for critical features: bore diameter, face mill, drill and tap.
  5. For each machining op, select machine (3-axis HAAS, 5-axis DMG) and enter cycle time.
  6. Review the combined cost breakdown — casting typically represents 40–65% of total cost.
Tips
  • Move machining tolerances into the casting wherever possible — every machined feature adds £0.30–1.50 per part.
  • For sealing faces, minimum machining (0.5 mm stock removal) is faster and cheaper than full face milling.
  • Specify T5 instead of T6 where mechanical properties allow — saves 3–4 h heat treatment cycle time.
Common Mistakes
  • Costing the casting and machining separately — use Cast+Machine tab for accurate combined tooling amortisation.
  • Forgetting impregnation for HPDC housings — porosity in HPDC is common; impregnation is mandatory for fluid-tight parts.
  • Under-estimating fixturing time — a complex 5-axis fixture changeover can add 12–25 min per batch.
Overview — Assembly BOM Rollup

The Assembly BOM model computes total assembly cost from a component list plus assembly operations. It supports multi-level BOM rollup, sub-assembly costing, and BOM import from CSV. Use this for any part where you are buying sub-components and adding value through assembly.

Inputs Required
  • BOM lines: part number, description, unit cost (£), quantity per assembly
  • Assembly operations: operation name, machine/station type, cycle time (h), OEE, Manning
  • Sub-assembly costs (nested assemblies)
  • Annual volume and region
Step-by-Step Workflow
  1. Select Assembly BOM tab or ⊕ New Costing → Assemblies tile.
  2. Add BOM lines: enter each component with unit cost and quantity. Use Import CSV for large BOMs.
  3. Add assembly operations: torque tighten, press fit, clip, weld, test. Enter cycle time per operation.
  4. For sub-assemblies, enter the sub-assembly cost as a BOM line item.
  5. Set overhead %, margin %, and annual volume.
  6. Click Calculate. The model sums BOM cost + assembly labour + machine cost + overhead + margin.
Tips
  • For manual assembly stations, use Manning = 1.0 and set machine cost = 0 (labour-only station).
  • Robotic assembly: Manning = 0.1–0.25 but machine hourly rate is £45–120/hr vs. £18–35/hr manual.
  • CSV import saves time for 20+ component BOMs — download the template from the Import button.
Common Mistakes
  • Forgetting fasteners and consumables (adhesives, lubricants) — these are often 2–5% of total BOM cost.
  • Using supplier quoted sub-assembly costs without should-cost validation — always cost critical sub-assemblies independently.
  • Ignoring end-of-line test cost — functional test at 30–90 s per assembly adds £0.20–0.80/part at UK labour rates.
